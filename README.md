nmap-ncat-static-builds
=======================

Description
------------
The great "nmap" and "ncat" tools, from the [Nmap project](https://nmap.org).

The building workflow is [here](https://github.com/mycodedoesnotcompile2/nmap-ncat-static-builds/blob/main/.github/workflows/main.yml).

See the [Github release page](https://github.com/mycodedoesnotcompile2/nmap-ncat-static-builds/releases) of this repository to access prebuilt binaries for different architectures (x64, arm64) and different libc (GNU, musl).
